1 #Config Configuration

Stylesheet configuration

Carapace is built on the Scarab Sass utility framework. This document outlines Carapace's default Scarab configuration. For more information on configuration for the Scarab framework, refer to Scarab's documentation.

To replace the default configuration or define new custom variables, use the set() mixin.

Source: theme/default/config/_.scss, line 1

1.1 #Config.Baseline Baseline

Configure the global baseline value

@include set-default( baseline, rem(16) );

The stylesheet's baseline is responsible for the sizing of most element and components. You can declare multiples of your baseline in properties throughout your stylesheet using Scarab's bl() mixin.

It is recommended to use rem units for the baseline so that the UI scales with the user's browser font-size setting.

Source: theme/default/config/baseline.scss, line 1

1.2 #Config.Breakpoints Breakpoints

Configure global breakpoints

@include set-default( breakpoints, (
    tiny:   em(480),
    small:  em(640),
    medium: em(960),
    large:  em(1440),
    huge:   em(1920)
) );

The breakpoints configuration map is used in conjunction with Scarab's breakpoint(), type-scale(), query() and responsive() mixins.

Adjust these breakpoints as required by your design. You may set a different number of breakpoints than what is provided by default and use them throughout your project.

Source: theme/default/config/breakpoints.scss, line 1

1.3 #Config.Durations Durations

Configure global durations

@include set-default( durations, (
    pulse:   0.05s,
    short:   0.1,
    medium:  0.25s,
    long:    0.8s,
    gradual: 1.6s

@include set-default( durations, default, duration(short) );

Configure durations for consistent transition and animation duration values throughout your project. Use Scarab's duration() getter function to return the value of a named duration.

Carapace sets the default duration to short.

1.9 #Config.Breakpoint-Modifiers Breakpoint modifiers

Configure breakpoint-based class modifiers

@include set-default( breakpoint-mods, (
    tiny,
    small,
    medium,
    large,
    huge
) );

Define a list of breakpoint-mods that Carapace will use to generate breakpoint "hooks" for each layout and helper class.

An example of a breakpoint "hook" is the .align-right@medium class, which will only apply the .align-right class to an element if the viewport is >= the width of the medium breakpoint.

Source: theme/default/config/breakpoint-mods.scss, line 1

Configure size-based class modifiers
